What is FlashLedger?

FlashLedger is a production-grade, low-latency order matching engine that replicates the core infrastructure of a modern crypto exchange. It demonstrates enterprise-level systems engineering with microsecond-level matching, real-time streaming analytics, and AI-driven market insights.

Order Matching

Order matching in microseconds using price-time priority heap algorithm

Real-Time Streaming

Apache Kafka + PySpark for live market microstructure analysis

Machine Learning

LSTM price prediction & LightGBM market regime classification

LLM Integration

Groq AI agent that generates natural-language trading commentary

Why FlashLedger?

This project demonstrates full-stack systems engineering expertise across multiple domains:

  • Trading Systems: Real-time order matching with microsecond latency and in-memory data structures
  • Distributed Systems: Kafka streaming pipelines, asynchronous FastAPI, WebSocket broadcasting
  • Machine Learning: Feature engineering on live data, model inference at scale, LLM orchestration
  • DevOps: Multi-stage Docker builds, containerized deployment on HF Spaces, git-based CI/CD
  • Frontend: React 18 real-time dashboard with live charts, trade feeds, and AI insights

How to Experience the Demo

  1. Open the Live Demo: Click the button above to launch the interactive dashboard
  2. View Order Book: See real-time order matching in the central order book widget
  3. Run Demo Scenarios: Click "Demo Controls" to run 5 different agent types across 5 market regimes
  4. Watch ML Predictions: View LSTM price predictions updating in real-time
  5. Check Market Insights: LightGBM recommendations with Groq LLM-generated commentary
  6. AI Agent Trading: Run the autonomous agent that generates orders via Groq Llama 3.3
  7. Live Chat: Interact with the Groq LLM chatbot for market analysis

Architecture Highlights

Multi-Layer Real-Time Processing:

  • Request Layer: FastAPI REST + WebSocket endpoints
  • Matching Engine: In-memory heapq with price-time priority
  • Streaming Layer: Kafka topics → PySpark transformations
  • ML Layer: Real-time feature engineering & model inference
  • LLM Layer: Groq API for commentary & autonomous trading
  • Frontend Layer: React dashboard with live updates
